# Night Of Nights

### What is it?
Night of Nights is a piece of harmless "malware", designed to annoy you. It is written in VB.NET, targeted at Windows.

### Stages
1. **Fake BSoD**: This is in "bsod.vb", and will last for about 10 seconds. At this point we start the music!
2. **Floater**: This is in "floater.vb", and will last for about a minute (or when the form is caught!), and will display a form that floats around the screen really fast!
3. **tbd**
4. **Mouse wobble + Keyboard Spam + Notepad + 3 Floaters**: This is in "stage3.vb", with each component being in the "Stage 3" filter in the solution. The mouse will wobble as notepad opens and sends keyboard spam, while opening 3 floaters.
5. **Touhou Mode**: Forms go full on Touhou mode.
